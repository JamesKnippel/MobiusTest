# Installation Requirements

1) We'll need Angular's CLI to ng build thus:
``` npm install -g @angular/cli ``` 

2) Run the Ruby Backend 

```
    from /Mobius 

    cd rubyPalApi`

    bundle install

    rails s

 ```

3) Run the front end 
  ``` from /rubyPalApi  
      
      cd .. 

      cd rubyPal

  ```


4) Install Dependencies and Serve Files

``` from /rubyPal

    npm install

    ng build

    node server/server.js

  ```

5) visit `localhost:1337`
